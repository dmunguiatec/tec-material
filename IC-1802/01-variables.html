Variables

Introducción a la programación

Variable

Una variable es una etiqueta que se refiere a un valor.

Una variable es el nombre de una casilla que contiene un valor, la casilla puede ser vaciada, su valor puede ser consultado y reemplazado.

Variables globales

Son accesibles dentro y fuera de funciones

Variables locales

Accesibles únicamente dentro de funciones

Acceso a variable global y local


global_var = 'foo'
def ex1():
    local_var = 'bar'
    print global_var
    print local_var

ex1()
print global_var
print local_var  # this gives an error

Cómo no modificar una variable global


var = 'foo'
def ex2():
    var = 'bar'
    print 'inside the function var is ', var

ex2()
print 'outside the function var is ', var

Cómo modificar una variable global


var = 'foo'
def ex3():
    global var
    var = 'bar'
    print 'inside the function var is ', var

ex3()
print 'outside the function var is ', var
